## Symbols

| Symbol | Meaning |
|--------|---------|
| вң“ | Pass / Verified |
| вң— | Fail / Issue found |
| в—Ҷ | In progress |
| в—Ӣ | Pending |
| вҡЎ | Auto-approved |
| вҡ  | Warning |

## Confidence Bar

```
в–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–Ҳв–‘в–‘ 85/100
```

Block characters showing confidence score. 10 characters wide.
- `в–Ҳ` = filled (score / 10)
- `в–‘` = empty (remaining)

## Footer

Every output ends with:
```
github.com/spsk-dev/code-review
```
